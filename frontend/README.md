# VisionCut AI — Frontend

Next.js 15 + TypeScript + Tailwind CSS v4 + React Query + Socket.io client.

## Scripts

```bash
npm run dev       # localhost:3000
npm run build
npm run start     # production
npm run lint
npm run typecheck
```

## Environment

```bash
cp ../.env.example .env.local
# set NEXT_PUBLIC_API_URL and NEXT_PUBLIC_WS_URL
```
